## 📖 项目简介

**华智云视 (HuaZhi CloudView)** 是一款专为高可靠性、应急指挥及专业会议场景设计的移动端应用。

它集成了实时高清视频回传、GIS 地理位置动态监控及双工语音对讲功能，旨在为一线人员与指挥中心提供无缝、低延迟的信息交互体验。

## 🌟 核心特性

- **1. 智能化身份验证与接入**：安全高效的鉴权机制，保障指挥链路安全。
- **2. 全能会中/回传体验**：支持高清视频实时回传，适应复杂网络环境。
- **3. 华智 AI 助手**：内置智能辅助，提升决策效率。
- **4. GIS 地理位置监控**：实时追踪人员位置，动态调度资源。
- **5. 专业语音对讲**：高保真双工语音，确保指令清晰传达。

## 🎨 视觉规范 (UI/UX)

应用采用 **简约多彩 (Minimalist & Vibrant)** 与 **专业深色 (Professional Dark)** 双重视觉逻辑，兼顾美观与夜间作战需求。

| 设计维度 | 细节说明 |
| :--- | :--- |
| **质感** | 全局应用 **磨砂玻璃 (Glassmorphism)** 效果，营造现代通透感。 |
| **层级** | 左侧组织架构采用彩色 **L1/L2/L3** 等级标识，右侧采用统一的卡片化资源列表。 |
| **反馈** | 关键状态（如“回传中”、“对讲中”）配有**绿色呼吸灯**动效，操作按钮具备微光触觉反馈。 |

## 🚀 快速开始

只需简单四步即可接入指挥系统：

1.  **配置服务器** 点击登录页左上角 `⚙️` 图标，进入“高级设置”输入服务器 IP 及端口。
2.  **测试连接** 点击“测试连接”按钮，确认显示绿色勾选 ✅ 及“连接成功”动画。
3.  **进入频道** 选择对应的登录节点及账号，点击“登录”进入仪表盘。
4.  **开启协作** 通过底部导航栏在 `资源`、`会议`、`通讯录` 间快速切换。
