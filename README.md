华智云视 (HuaZhi CloudView) - 专业移动会议与监控指挥系统
LicensePlatformTheme

华智云视 是一款专为高可靠性、应急指挥及专业会议场景设计的移动端应用。它集成了实时高清视频回传、GIS 地理位置动态监控、双工语音对讲以及基于 Gemini 引擎的 华智AI 助理，为用户提供沉浸式的远程协作体验。

🌟 核心特性
1. 智能化身份验证与接入
多维度登录：支持传统账号密码及手机验证码快速登录。
节点与服务器自治：内置“节点选择”下拉菜单（磨砂玻璃质感），支持自定义服务器 IP 与端口合并输入，配备 “测试连接”动画反馈 系统。
2. 全能会中/回传体验
多比例布局：支持单画面、3画面、4画面（含特殊 16:9 与 1:1 混合布局）及 9画面宫格显示。
沉浸式视频：强制 16:9 画面居中显示，支持全屏填充模式，消除视频黑边。
手感化控制栏：悬浮式圆形工具栏，集成 复合对讲麦克风、一键截图、摄像头翻转及手机 UI 翻转按钮。
3. 华智AI 助手 (AI Assistant)
双窗体悬浮交互：在不中断视频的情况下，右侧边缘吸附 AI 悬浮窗。
核心功能：实时 会议转写、自动生成 会议纪要 以及动态 会议导图，支持数据一键复制与本地下载。
4. GIS 地理位置监控
高精度定位：地图中心实时显示位置图钉，动态标注“人员名称 + 状态”。
数据可视化：醒目的双行经纬度（E: 蓝色 / N: 紫色）显示，支持背景对比度增强色块。
业务联动：支持从地图定位界面直接一键开启视频预览。
5. 专业语音对讲 (Intercom)
PTT (Push-to-Talk)：经典“按住说话”大按钮逻辑，配合实时动态音频波形图。
双主题弹窗：提供深色磨砂与纯净白色两种风格的对讲邀请模态框。
🎨 视觉规范 (UI/UX)
应用采用 简约多彩 (Minimalist & Vibrant) 与 专业深色 (Professional Dark) 双重视觉逻辑：

质感：全局应用磨砂玻璃 (Glassmorphism) 效果。
层级：左侧组织架构采用彩色 L1/L2/L3 等级标识，右侧采用统一的卡片化资源列表。
反馈：关键状态（如“回传中”、“对讲中”）配有绿色呼吸灯动效，操作按钮具备微光触觉反馈。
🛠️ 技术要求
iOS: 13.0+  iOS 代码：13.0+
Android: 9.0+  安卓 ：9.0+
网络: 建议 4G/5G 或 Wi-Fi 环境以保证 1080P 回传流畅。
后端支持: 需配套华智云控制台及信令服务器。
🚀 快速开始
配置服务器：点击登录页左上角齿轮图标，进入“高级设置”输入服务器 IP 及端口。
测试连接：点击“测试连接”，确认显示绿色勾选及“连接成功”动画。
进入频道：选择对应的登录节点及账号，点击“登录”进入仪表盘。
开启协作：通过底部导航栏在“资源”、“会议”、“通讯录”间快速切换。
